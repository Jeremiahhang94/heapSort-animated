- Heap Sort -

A comparision-based sorting algorithm.
The first step is to build a heap out of the data like a binary tree.
In an typical array, the root note would be stored at index 0, with the child at 2*i+1 and 2*1+2

Afterwhich the tree have been constructed, the second step would commence such that the largest element would be swaped with the smallest element and there after removed from the heap into the array, everytime a swap occurs, the heap would be updated again to maintain the heap. Once all the object have been removed from the heap, the sorted array is thus formed.
